1) Install dependency (required)

  composer install
- This will create vendor/autoload.php and install PHPMailer.

2) Enable Google account prerequisites
- Log into miuletdaniel@gmail.com.
- Google Account > Security.
- Turn ON 2-Step Verification.
- Create App Password:
  - Security > App passwords
  - App: Mail
  - Device: Other (example: AstroTarot Website)
  - Copy and save the 16-character password.

3) Configure environment variables on hosting/server
- SMTP_HOST=smtp.gmail.com
- SMTP_PORT=587
- SMTP_ENCRYPTION=tls
- SMTP_USERNAME=miuletdaniel@gmail.com
- SMTP_PASSWORD=YOUR_16_CHAR_APP_PASSWORD
- SMTP_FROM_EMAIL=miuletdaniel@gmail.com
- SMTP_FROM_NAME=Astro Tarot
- SMTP_BCC_EMAIL=miuletdaniel@gmail.com
- Alternative for local run: create backend/.env (copy from backend/.env.example) and fill values there.

7) Troubleshooting
- "Dependintele lipsesc. Ruleaza composer install." => run composer install in project root.
- "Lipseste SMTP_PASSWORD..." => SMTP_PASSWORD not set in environment.
- SMTP timeout/connection errors => hosting firewall blocks smtp.gmail.com:587 or 465.
- 500 in PowerShell Invoke-RestMethod: print response body to see exact PHPMailer error text.
